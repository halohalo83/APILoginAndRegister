***API LOGIN VÀ REGISTER ***

I. Các chức năng chính:
1. Đăng nhập
	- Yêu cầu nhập Username và Password không được để trống
	- Nhập đúng username và password đã đăng ký
	- username, password và email được lưu trong database
	- Nếu đăng nhập đúng, điều hướng trang về "Home", nếu không sẽ trở lại màn hình đăng nhập
2. Đăng ký
	- Bấm vào "Click to register" để điều hướng sang trang đăng ký
	- Có 4 trường username, email, password,repeat password
	- Trường Username cần phải nhập đúng định dạng (bao gồm cả chữ và số)
	- Trường email cần phải nhập đúng định dạng (abc@example.abc)
	- Trường password và repeat password phải trùng nhau
3. Lưu đăng nhập
